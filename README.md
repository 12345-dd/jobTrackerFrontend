# Job Analytics — Frontend

A full-stack MERN application that helps users organize their job search, track application progress, and analyze where they are getting stuck — such as HR rounds or technical interviews.

**Live Demo:** [https://jobanalytic.netlify.app](https://jobanalytic.netlify.app)  
**Backend Repo:** [https://github.com/12345-dd/jobTrackerBackend](https://github.com/12345-dd/jobTrackerBackend)

---

## Features

- **JWT-based Authentication** — Secure signup & login
- **Job Application Tracker** — Add, update, and manage applications
- **Analytics Dashboard** — Visual insights on your job search progress
- **Resume Skill Analysis** — Get skill suggestions based on job requirements
- **Protected Routes** — Only authenticated users can access their data

---

## Getting Started (Local Setup)

### Prerequisites
- Node.js v18+
- npm or yarn
- Backend server running (see backend repo)

### 1. Clone the Repository
```bash
git clone https://github.com/12345-dd/jobTrackerFrontend.git
cd jobTrackerFrontend
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Run the App
```bash
npm run dev
```

> App runs on `http://localhost:5173`
